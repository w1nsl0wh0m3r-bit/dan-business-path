A living dashboard for comparing acquisition, franchise, and DIY opportunities. Route services usually belong in buy/franchise. Multi-unit consumer concepts are where DIY can create the playbook and capture the upside.

Entry mode framework

Buy / acquire

Use when the scarce asset already exists: customers, technicians, licenses, reviews, route density, local trust, or cash flow.

Route servicesHome systemsCustomer books
  • Fatal risk: buying an owner-dependent job.
  • Key gate: field leader + customer file quality.

Franchise

Use when the system is worth royalty drag: training, compliance, purchasing, site selection, territory design, benchmarking, or real lead gen.

System valueFDD diligenceFranchisee calls
  • Fatal risk: paying royalties for a logo and generic marketing.
  • Key gate: unit economics after royalty/ad fund.

DIY / build

Use when brand and playbook creation are the upside: taste, content, location, community, product design, and repeatable unit ops.

Multi-unitRetailEducation
  • Fatal risk: lease/payroll before demand proof.
  • Key gate: low-capex validation sprint.

Current recommendations

Top acquisition sprint

Drainage / yard water management + irrigation as the affluent exterior systems wedge. Under-picked enough to be interesting, route/service dense enough to buy.

Top DIY sprint

Premium candy or Kids STEM / AI maker lab. Very different shapes: consumer brand vs education/community. Both can be tested before permanent infrastructure.

Top franchise sprint

Only where FDD proves system value. Metal Supermarkets and select home-services systems deserve diligence; generic brand-only franchises do not.
